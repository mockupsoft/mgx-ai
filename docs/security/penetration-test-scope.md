# Penetration Test Scope

## Objective

Validate security posture of the production deployment and identify exploitable vulnerabilities.

## Scope

- API endpoints (REST + WebSocket)
- Authentication and authorization
- Workspace isolation
- Secret handling
- Outbound integrations (SSRF)

## Out of Scope

- Social engineering
- Physical security

## Duration

- 1 week

## Deliverables

- `docs/security/penetration-test-report.md`
- `docs/security/pentest-findings-remediation.md`
